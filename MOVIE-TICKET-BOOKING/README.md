# 🎬  Movie Ticket Booking API

This project is a **FastAPI-based Movie Ticket Booking System** built for the FastAPI Internship Final Project.  
It allows users to browse movies, hold seats, confirm bookings, and manage tickets using REST APIs.

# 🚀 Features

## Day 1 – GET APIs
- Home route
- Get all movies
- Get movie by ID
- Movies summary
- Get all bookings

## Day 2 – POST + Pydantic Validation
- Booking request validation
- Field constraints
- Seat type validation
- Promo code support

## Day 3 – Helper Functions
- find_movie()
- calculate_ticket_cost()
- filter_movies_logic()

## Day 4 – CRUD Operations
- Add new movie
- Update movie
- Delete movie
- Duplicate check
- 404 handling

## Day 5 – Multi-step Workflow
Seat Hold → Confirm → Release

- Hold seats temporarily
- Confirm booking
- Release seats
- Convert hold to booking

## Day 6 – Advanced APIs
- Movie search
- Sorting
- Pagination
- Combined browse endpoint
- Booking search/sort/page


Everything is wrapped up and applied on this project
